Prownloader
===========

Utilitário para download automático das vídeo-aulas!

### Tecnology
- Java 7
- Swing (windows/linux desktop app)
- Hibernate
- JPA2
- HSQLDB
- MAVEN
- Web Client
- Linux and Windows
